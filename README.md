# Best Amount Enterprise Website

A professional, responsive corporate website for Best Amount Enterprise, specializing in mobile money transfer services, banking, and general trading.

## Project Structure

```
BEST-AMOUNT/
├── index.html          # Home page
├── about.html          # About Us page
├── services.html       # Services page
├── contact.html        # Contact page
├── css/
│   └── main.css        # Main stylesheet
├── js/
│   └── script.js       # JavaScript for interactivity
└── images/             # Directory for images and icons
```

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Modern UI**: Corporate styling with deep blue, white, and gold accents
- **Interactive Elements**: Sticky navigation, mobile menu, form validation
- **Accessibility**: Semantic HTML, keyboard navigation support
- **Fast Loading**: Optimized CSS and minimal JavaScript

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Vanilla JavaScript

## Hosting Instructions

### Option 1: GitHub Pages (Free)
1. Create a GitHub repository
2. Upload all files to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch" and choose `main` branch
5. Your site will be available at `https://yourusername.github.io/repository-name/`

### Option 2: Netlify (Free)
1. Sign up for a Netlify account
2. Drag and drop the project folder to the Netlify dashboard
3. Your site will be deployed instantly with a custom URL

### Option 3: Traditional Hosting
1. Purchase web hosting (e.g., Hostinger, Bluehost)
2. Upload files via FTP to the public_html directory
3. Ensure the hosting supports static HTML sites
